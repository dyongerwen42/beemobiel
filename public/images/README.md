# Images Directory

Plaats hier al je afbeeldingen voor de website.

## Gebruik in Next.js

Je kunt afbeeldingen gebruiken met het Next.js Image component:

```tsx
import Image from 'next/image'

<Image 
  src="/images/logo.png" 
  alt="Logo" 
  width={200} 
  height={200}
/>
```

Of met een normale img tag:

```tsx
<img src="/images/logo.png" alt="Logo" />
```
